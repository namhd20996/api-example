Dự án này là một trang web bán hàng được xây dựng bằng [Spring Framework], với các tính năng sau:

- Hiển thị danh sách sản phẩm
- Thêm sản phẩm vào giỏ hàng
- Quản lý giỏ hàng
- Thực hiện đơn hàng và thanh toán
